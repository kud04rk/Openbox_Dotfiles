# Openbox shortcuts

## Windows
### Cycle Windows
```
super/windows + Shift + up/down/left/wight arrows
```
### Snap windows
```
super/windows + up/down/left/wight arrows
```
### Send window to desktop
```
super/windows + Shift + 1 2 3 ..8
```
### Show title/Window menu
```
super/windows + space
```
### Toggle window decorations
```
super/windows + t
```
### Full screen
```
super/windows + f
```
### Toggle maximise 
```
super/windows + x
```
### Minimize
```
super/windows + z
```
### Close window
```
super/windows + c
```
### Show desktop Toggle
```
super/windows + d
```
